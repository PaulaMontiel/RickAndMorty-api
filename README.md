# Rick & Morty Episodes – Jetpack Compose UI

## Introducción

Este proyecto corresponde a la implementación de la interfaz visual de un listado de episodios de **Rick & Morty** utilizando **Jetpack Compose**.

El proyecto base ya contaba con la lógica necesaria para consumir la API pública de Rick & Morty y obtener la información de los episodios.

El objetivo de este trabajo fue **implementar el diseño visual del listado**, manteniendo intacta la lógica existente del **ViewModel** y del **consumo de la API**.

API utilizada:  
https://rickandmortyapi.com/
